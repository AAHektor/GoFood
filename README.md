# GoFood

GoFood is a food delivery MVP built with a React frontend and an ASP.NET Core backend.
For now only mobile responsive.

### OBS!!!
- Only mobile responsive for now

## What it does
- Browse restaurants and view menus
- Add items to cart and review order
- Create and manage orders through a REST API

## Tech Stack
- Frontend: React + Vite
- Backend: ASP.NET Core + Entity Framework Core
- Database: SQL via EF Core migrations

## Future Features
- Responsive UI improvements for all screen sizes
- Mobile app packaging with Capacitor
- User authentication with Entity Framework + Gmail sign-in
- Restaurant owner dashboard (manage menu, orders, availability)
- Checkout and payments with Stripe
- Order history and saved addresses
- Live order status tracking (preparing, on the way, delivered)
